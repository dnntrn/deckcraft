# Studio Theme

Clean, universal presentation theme. Swiss/International design — minimal, functional, beautiful through restraint.

## When to use

- All-purpose: works with any brand, any topic, any audience
- Light or dark mode
- Technical talks, business presentations, training, pitch decks
- When you want the content to speak, not the theme

## Design principles

- Generous whitespace (80px padding)
- System font stack (renders cleanly everywhere)
- Thin accent border on titles (4px left-stripe)
- Clean code blocks with subtle shadow
- Neutral Mermaid diagrams with accent edges
- Section dividers with serif display text
- No animations by default

## Customization

All colors are CSS custom properties. Override `--slidev-accent`, `--slidev-canvas`, `--slidev-foreground`, etc. to match any brand.

## Files

- `styles.css` — Full stylesheet for Slidev
- `setup.ts` — Font loading and Mermaid theme configuration
- `examples.md` — Sample slides showing each layout
